# NutriBalance_NLP

<pre class="!overflow-visible"><div class="contain-inline-size rounded-md border-[0.5px] border-token-border-medium relative bg-token-sidebar-surface-primary dark:bg-gray-950"><div class="overflow-y-auto p-4" dir="ltr"><code class="!whitespace-pre hljs language-markdown"># NutriBalance

NutriBalance is a project designed to help users track and manage their food consumption by parsing input strings like `"I ate 500 gram of apple, 2g of sugar, and 9g of bread"` into a structured format, such as:

```json
{
  "500g": "apple",
  "2g": "sugar",
  "9g": "bread"
}
</code></div></div></pre>

## Features

- Parse user input into structured data.
- Extract quantities and corresponding food items.
- Designed for extensibility and modularity.
